RailgGirlsちょこっと説明資料
=========================

nanoc + haml で作っています。  

### 前提条件  

* rubyがインストールされていること  

### とりあえず動かしてみる  

1. コマンドを実行： git clone https://github.com/inoh/RailsGirls.git  
2. コマンドを実行： cd RailgGirls/slide  
3. コマンドを実行： bundle install
4. コマンドを実行： nanoc aco  
5. ブラウザを起動して http://localhost:3000/ にアクセス
