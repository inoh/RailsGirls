HTMLとRuby(ERB)の違い

Ruby(ERB)
<% 〜 %> <- 囲まれた箇所がRubyからHTMLに変換される
<%= @idea.name %> # => ステッカー
    <%= f.label :name %> 
 => <label for="idea_name">Name</label>
    <%= f.text_field :name %> 
 => <input id="idea_name" name="idea[name]" type="text" />

HTMLとRuby(ERB)の違い

HTML
<% 〜 %>以外 <- 囲まれていない箇所についてはそのまま出力される
<h1>Listing ideas</h1>
 => <h1>Listing ideas</h1>
