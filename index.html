HTMLとRuby(ERB)の違い

HTML ※Webブラウザにいろいろ表示するための言語
<% 〜 %>以外 <- 囲まれていない箇所についてはそのまま出力される
<strong>Name:</strong>
 => Name:

HTMLとRuby(ERB)の違い

Ruby(ERB) ※RubyでHTMLを作るもの
<% 〜 %> <- 囲まれた箇所がRubyからHTMLに変換される
<%= @idea.name %>  => ステッカー
    <%= f.label :name %> 
 => <label for="idea_name">Name</label>
    <%= f.text_field :name %> 
 => <input id="idea_name" name="idea[name]" type="text" />

routesの書き方

「config/routes.rb」を修正

「ideasの一覧をください」 => get 'ideas'
    「ideasのidをください」 => get 'ideas/:id'
    「ideasを新しく作ってください」 => post 'ideas'
    「ideasのidを更新してください」 => put 'ideas/:id'
    「ideasのidを削除してください」 => delete 'ideas/:id'

routesの書き方

resources :ideas

「ideasの一覧をください」 => get 'ideas'
    「ideasのidをください」 => get 'ideas/:id'
    「ideasを新しく作ってください」 => post 'ideas'
    「ideasのidを更新してください」 => put 'ideas/:id'
    「ideasのidを削除してください」 => delete 'ideas/:id'
    「ideasの新規登録画面をください」 => get 'ideas/new'
